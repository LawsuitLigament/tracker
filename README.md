
# tracker, an easy way to track your attendance.

It's an app to track your attendance so that you never fall behind on your classes. Some schools/universities/colleges require a certain minimum attendance criteria, this app lets you set that criteria and also lets you know how many classes you can miss or how many you need to take to get back to criteria.
